# Awesome Event

## サイト概要
勉強会などの予定を作成し募集をかける。また、それを見た人が参加表明をできる

### サイトテーマ
デザインを整えて閲覧しやすいサイトにする

### テーマを選んだ理由
ユーザーが使いやすいサイトにするため

### ターゲットユーザ
イベントを開催したい人。またそのイベントに参加したい人

### 主な利用シーン
イベントの作成と参加
